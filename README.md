# Symfony Vue Webshop

This project is a webshop sandbox built with Symfony, Vue, Tailwind CSS, and Webpack Encore.

It is meant as a training ground for experimenting with modern frontend/backend workflows, reusable Vue components inside Twig templates, REST APIs, repository and storage adapters, and simple ecommerce features without needing a database-heavy setup.

## Features

- Product catalog with category pages and product detail pages
- Product listing pagination
- Vue-powered cart interactions
- REST-based checkout cart API
- Cart page with quantity changes, item removal, tax, and order summary
- Wishlist API and Vue wishlist interactions
- Checkout flow pages for cart, shipping, payment, and summary
- Responsive header, footer, mobile offcanvas menu, and hero banner
- JSON-backed repositories with adapter-style clients
- Local image storage with an adapter structure for S3
- Tailwind-based styling and Webpack Encore asset builds

## What Is Aperture?

Aperture is the fictional company from the Portal games.

I used Aperture as a theme to give this project an identity and make it more fun for myself while building and learning. The branding gives the webshop some character instead of feeling like a generic demo shop.
